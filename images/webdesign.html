Introduction to Scratch
Age: 9 and up Prereq: Scratch experience
ScratchÂ® is a programming language that was invented by the MIT Media Lab. Through the creation of interactive stories, animations, games, music and art, students will develop a foundation of programming concepts that will prepare them for higher level programming language in the future. Students will also learn to think creatively and reason systematically.

Intermediate Scratch Programming **NEW
Age: 9 and up                                Prereq: Scratch experience 
Students will code more elaborate programs, and explore advanced topics such as the external device extensions, byob (build your own blocks), integration with webcam and more. Under the guidance of the teacher, students will use of their newly acquired knowledge to build their innovative projects. 

Basic Web Design (previously HTML Programming)
Age: 9 and up                                 Prereq: Comfortable with typing
This course teaches the students how to build a web site using HTML (Hyperlink Text Markup Language) and CSS (Cascaded Style Sheets). Students will earn about links, texts, images, styles and various elements of the language of HTML and CSS. The class will also discuss the internet, domain names, web server, web browser, internet service provider, tools to build websites, as well as guidelines on the contents of websites.Concepts will be explained, examples will be provided, and activities will be assigned to reinforce the understanding of the concepts. The final project from the class will be a live website. Students will have access to the HTML and CSS files on the website for a year.

Introduction to Python Programming  ** NEW
Age: 9 and up                                 Prereq: Comfortable with typing
You will be coding in Python to create interesting programs from jokes, games (ie Hangman, Reversi) to artificial intelligence. Python has a gentle learning curve while still being a serious language that is used by programmers professionally. Student will learn to work with Python in its interactive mode and through the use of scripts. The software will be available for free.
